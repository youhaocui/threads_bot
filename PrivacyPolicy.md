# 隱私權政策

本隱私權政策說明 **TaiwanAlertBot**（以下稱「本服務」）如何蒐集、使用、儲存及保護使用者的資料。使用本服務即表示您同意本政策的內容。

## 一、資料蒐集
本服務僅蒐集並使用以下類型的資料：
- **必要的 API 金鑰與憑證**：例如中央氣象局 API 金鑰、Meta Threads API Token，用於存取官方資料與發佈訊息。
- **天氣與警特報資料**：來自中央氣象局公開資料平台（CWA Open Data）。
- **程式執行紀錄**：如上次發文的內容摘要，用於判斷是否需要再次發文。

> 本服務 **不會**蒐集使用者的個人身份資訊（如姓名、電話、地址）。

## 二、資料使用
蒐集的資料僅用於：
- 自動化比對天氣與警特報更新。
- 發佈最新警特報或天氣摘要至指定平台（Meta Threads）。
- 測試模式下，僅在終端機列印訊息，不會對外傳送。

## 三、資料儲存
- 本服務會在伺服器或執行環境中儲存「上次發文內容」與「縣市摘要紀錄」，以避免重複發文。
- 所有 API 金鑰與憑證均透過環境變數或 `.env` 檔案管理，不會硬編碼在程式中。

## 四、資料分享
- 本服務 **不會**將使用者資料出售、出租或分享給第三方。
- 唯一的例外是依照法律要求，或為了保護本服務的合法權益。

## 五、資料安全
- API 金鑰與憑證均採用環境變數管理，避免外洩。
- 程式僅存取必要的公開資料，不涉及敏感個資。

## 六、政策更新
本政策可能因服務調整或法規更新而修改。更新後的版本將公布於專案頁面或官方網站。

## 七、聯絡方式
若您對本政策有任何疑問，請透過專案維護者提供的聯絡方式與我們聯繫。
